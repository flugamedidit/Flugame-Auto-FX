3D TEXT POPUP LOGIC

═══════════════════════════════════════



Every text asset should be built using this logic:



1. PRIMARY 3D WORDMARK OBJECT



The selected text must be the dominant visual element.



It should feel like a custom physical 3D wordmark object, not flat text, not a typed font preset, and not a basic title overlay.



The wordmark should feel modeled, sculpted, lit, and rendered like a premium 3D title made by a motion designer in Cinema 4D / Octane-style 3D design.



The text should usually include:



- deep extrusion

- thick bevels

- visible sidewalls

- sculpted letter faces

- custom street / urban / rap-video font sensibility

- expressive letter relationships

- interlocking or overlapping forms when appropriate

- realistic highlights

- realistic cast shadows

- shadows from letters onto other letters

- reflections from letters onto other letters

- perspective matching the shot

- material that reacts to the scene lighting



The wordmark should feel like one cohesive sculptural object, not separate typed words sitting neatly beside each other.



2. LETTER INTERACTION



The letters should interact with each other as parts of one 3D composition.



Avoid:



- perfectly formal text layout

- plain stacked words

- evenly spaced corporate typography

- clean subtitle-like alignment

- standard typed lettering

- conservative title-card design

- generic Times New Roman / office-style font behavior

- simple block font with bevel only



Prefer:



- overlapping letters

- tucked letterforms

- interlocking shapes

- slight collisions

- varied depth between letters

- stretched or compressed letter shapes

- exaggerated curves, hooks, tails, or sharp edges

- cast shadows between letters

- glow caught inside letter gaps

- sculpted asymmetry

- custom urban wordmark shaping



The text should feel like 3D lettering art, not formal text.



3. CONTEXTUAL SUPPORT OBJECTS



The popup should usually include 1 to 3 contextual support elements when the lyric, phrase, or mood gives a clear direction.



Support elements should emerge from behind, beside, underneath, through, or around the wordmark.



They should feel integrated into the 3D title sculpture, not pasted on as random decorations.



Examples:



- money / flex themes: cash bands, card edges, money corners, diamond glints, jewelry-like chrome pieces

- weapon / danger themes: bullets, shell casings, target marks, metal fragments, sparks

- pressure / aggression themes: cracks, compression marks, impact shards, smoke, bent metal

- luxury themes: diamonds, polished chrome arcs, gold accents, reflective jewelry fragments

- motion themes: speed slashes, metallic trails, angled shards, curved light arcs

- pain / dark themes: red liquid accents, torn shadow forms, black smoke, dripping shapes

- cold themes: ice shards, frosted edges, crystalline fragments, pale blue rim light

- heat themes: ember glow, heated metal cuts, smoke plumes, orange rim light



Do not add random props.

Do not add too many props.

Do not add support objects that compete with the text.

Do not add people, faces, brands, logos, UI, or unrelated symbols.



Support objects must reinforce the text, lyric, mood, or scene context.



If the phrase clearly suggests flex, motion, pressure, aggression, luxury, money, cold, heat, pain, danger, or emotion, do not default to “no extra props.” Create restrained contextual support elements that belong to that idea.



4. BACKING ENERGY LAYER



The popup should have a backing layer behind the text to create separation, depth, and impact.



This can be:



- controlled glow

- smoke bed

- light bloom

- red/orange rim glow

- cyan edge glow

- liquid splash

- dark shadow plate

- energy halo

- sparks

- subtle lens-reactive aura

- gritty texture plate

- metallic reflection plate



The backing layer should wrap behind and around the wordmark, helping the 3D form feel embedded in the frame.



It should not become a random VFX explosion or a large cloudy blob unless the lyric or mood calls for that.



5. HERO TITLE COMPOSITION



The final result should feel like a mini 3D title composition, not just a text label.



Think:



hero sculptural wordmark + interactive lettering + contextual support objects + backing energy layer + clean readable silhouette.



It should not feel like:



- plain subtitles

- formal text

- a flat sticker

- a default font with bevel

- random poster art

- unrelated logo design

- a full scene replacement

PLACEMENT RULES

═══════════════════════════════════════



Place the 3D text popup as a bold center-frame hero graphic.

The popup should usually live in the middle of the frame, large and dominant, like a music-video title moment or 3D emblem dropped directly into the shot.

It may overlap non-critical areas of the subject or background if the composition looks intentional, but it should avoid completely destroying the face, eyes, mouth, or main performance read unless the user specifically asks for a face-obscuring effect.

Placement priority:

1. Center-frame hero composition
2. Middle screen space with the strongest visual impact
3. Slightly above or below center if it better fits the subject’s body/action
4. Integrated over the scene as a dimensional title object
5. Negative space only if center placement would ruin the shot

The popup should not feel tucked away in a corner. It should feel like the main graphic event of the frame.
